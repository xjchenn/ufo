To run the server (tested on linux):

1) Install pip: 

easy_install pip

2) Install virtualenv: 

pip install virtualenv

3) Create virtualenv: 

virtualenv venv
source venv/bin/activate

4) Install dependencies:

pip install -r requirements.txt

5) Run server:

make run

It is good practice to work with virtualenv in python, so perform step 3 before doing any development work
